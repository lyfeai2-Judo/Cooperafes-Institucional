# 03 — Estrutura do site (como tudo está organizado)

Este documento explica como o site está organizado por dentro.

A ideia é mostrar onde fica cada coisa e por que essa organização foi escolhida.

Nada aqui é por acaso.

---

## A pasta principal do site

Tudo que aparece para o público fica dentro da pasta:

site/

Essa pasta é a “caixa principal” do site.

Tudo que está fora dela não aparece na internet.

---

## Arquivo principal do site

Dentro da pasta `site/`, existe o arquivo:

index.html

Esse arquivo é a **porta de entrada** do site.

Quando alguém acessa o site, é esse arquivo que aparece primeiro.

Ele contém:
- o nome do projeto
- uma explicação simples do site
- links para os textos
- o status das fases do projeto

---

## Pasta de textos institucionais

Dentro da pasta `site/`, existe a pasta:

textos/

Essa pasta guarda os textos do site, como:
- Home
- O Observatório
- Contato
- Parceiros
- Galeria
- Livros

Esses textos ficam separados para facilitar:
- leitura
- revisão
- atualização futura

---

## Pasta de documentação

Dentro da pasta `site/`, existe a pasta:

docs/

Essa pasta guarda os documentos que explicam:
- como o site foi feito
- quais ferramentas foram usadas
- como repetir o processo em outros sites

Ela é importante porque transforma o site em um **material de aprendizado**.

---

## Por que essa estrutura é simples?

A estrutura foi pensada para ser:
- fácil de entender
- fácil de manter
- fácil de ensinar
- fácil de repetir

Quanto mais simples a estrutura,
menos problemas aparecem no futuro.

---

## O que pode crescer depois?

No futuro, podem ser adicionadas:
- novas páginas
- imagens
- melhorias visuais
- domínio próprio

Mas a base continuará a mesma.

---

## Resumo final

A estrutura do site é simples porque o objetivo é clareza.

Organizar bem no começo evita confusão depois.

Essa mesma estrutura pode ser usada em outros projetos,
mudando apenas os textos e o nome do site.
