# 01 — Visão Geral do Projeto

## O que é este site?

Este site é a base institucional do **Observatório Cultural de Sergipe**.

Ele foi criado para reunir:
- Informações institucionais
- Textos oficiais
- Materiais de referência
- Estrutura base para outros sites parceiros

Neste primeiro momento, o site é simples e direto, com foco em organização e clareza.

---

## Qual é o objetivo deste projeto?

O objetivo principal é:

- Colocar o Observatório no ar rapidamente
- Criar uma base sólida e organizada
- Permitir evolução futura sem retrabalho

Este site também funciona como o **“site-pai”**, servindo de apoio e divulgação para outros projetos e parceiros.

---

## Por que o site é simples?

Porque este é um projeto em fases.

A prioridade inicial foi:
- Funcionar
- Estar no ar
- Ser fácil de entender e manter

O design mais elaborado e os ajustes visuais virão depois.

---

## O que já foi conquistado?

✔️ Repositório organizado no GitHub  
✔️ Site publicado na Vercel  
✔️ Estrutura clara de pastas e arquivos  
✔️ Link público funcionando  

Este já é um avanço real e concreto.

---

## Mensagem importante

Este projeto prova que é possível criar e publicar um site do zero,
mesmo sem experiência técnica avançada,
desde que o processo seja bem documentado e feito passo a passo.
